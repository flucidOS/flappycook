# **Flappycook – Package Builder**

Flappycook is a package recipe builder used to generate
.pkg.tar.zst binary packages for the Flappy package manager.

> [!NOTE]
> **Flappycook** is currently under development as part of the **flucidOS package ecosystem**.
> It is used to build binary packages for flucidOS.
>
> The project is still in an early stage, and features may change as development progresses. Bugs and security issues are actively being monitored and addressed.


## **Build:**

`make` 

## **Example:**

`./flappycook build recipes/hello.recipe`

## **Output:**

`output/hello-2.12.pkg.tar.zst`
